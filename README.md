## Overview
- **Goal:** Self hostable service to track all human functions
- **Why:** Cool to see all the stats in one place / manage interconnected flows of data
- **Scope:** Human attributes / Lifestyle / Activities
- **Out of scope:** Medically-actionable data collection

## Highlights
- Simple UI
- Simple Use without losing functionality
- Easy integrations (as inputs or outputs) with other services
- Simple analytic tracking and visualisation
