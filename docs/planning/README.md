# Planning

## Summary
open-health-server will track [metrics](metrics.md) that pertain to human health, activities, lifestyles and choices. The server will be built around the idea of "users", where each server starts with a default "You" user, but allows the creation of new users to accomodate groups ( ex: a rowing team or a household ). open-health-server will be able to [integrate](integrations.md) into a multitude of services to allow for many different and personal flows of data.
